Leonarda Banić

Ja sam Leonarda. Rođena sam 2001. godine u Varaždinu. Završila sam Elektrostrojarsku školu Varaždin, smjer Web dizajner. Tijekom svog srednjoškolskog obrazovanja naučila sam raditi s mnogim programskim jezicima (Javascript, C++, PHP...), upoznala sam razne alate za obradu i izradu multimedijalnih sadržaja (Adobe Photoshop, Adobe Premier Pro, Adobe Illustrator Autodesk Maya, Unity...) te naučila kreirati web stranice. Pri izradi web stranica koristila sam alate Notepad++, Visual Studio Code i Adobe Dreamweaver, te CMS sustave Joomla i Wordpress.

Svoje sam obrazovanje nastavila na Fakultetu informatike u Puli.

U slobodno vrijeme volim igrati igrice, gledati anime, družiti se s prijateljima i ispijati kavu. Također volim fotografirati. Vidi slike u nastavku :-)
